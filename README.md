# Hockey Playground
Hockey quiz / YouTube video app built with jQuery.

## Using the app

* Click Play Now to test your hockey knowledge with a quiz
* Check the latest NHL Tonight videos from the gallery

### Coming Features

* More quizes
* Social sharing for quiz results

## Built With

* jQuery
* HTML5 BP / Sass
* Gulp
* Bourbon / Neat
